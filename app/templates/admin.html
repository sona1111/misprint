Update The Class Schedule

Updating the class schedule for a term

The class schedule must be populated before the schedule of classes in the app will function. Under normal circumstances this will only have to be done once per term. The input field below should be filled in with the name of the Selection choice for the page here. For example, "Spring 2016" (no quotes). This will attempt to update the schedule for this term. Please wait the whole time without closing the page, it can take a while. A message will appear once the process is done. Please run this only at non-peak times, as it takes a while blocking the main connection.
